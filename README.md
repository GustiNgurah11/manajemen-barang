Sistem Manajemen Barang adalah aplikasi **RESTful API** berbasis **Laravel** yang digunakan untuk mengelola data inventori barang secara terpusat. Sistem ini dilengkapi dengan **autentikasi JSON Web Token (JWT)** dan **log aktivitas pengguna** untuk meningkatkan keamanan serta monitoring penggunaan sistem.

Project ini dikembangkan sebagai bagian dari tugas mata kuliah **Pemrograman Web Service**.

---

## Fitur Utama

- Autentikasi pengguna menggunakan **JSON Web Token (JWT)**
- Manajemen data:
  - User
  - Kategori Barang
  - Barang
  - Supplier
  - Transaksi Barang
  - Pembayaran
- Relasi antar data inventori
- Log aktivitas pengguna (login, tambah, ubah, hapus data)
- RESTful API (GET, POST, PUT, DELETE)
- Pengujian API menggunakan **Postman**

---

## Teknologi yang Digunakan

- PHP 8+
- Laravel 10
- MySQL
- JSON Web Token (JWT)
- Postman
